# springboot-BMSystem
springboot+mybatis+mysql+thymeleaf+jquery构建的小型图书借阅管理系统  
**后端**  
1.springboot  
2.mybatis  
**数据库**  
1.mysql  
**前端**  
1.jquery  
2.jquery-validate  
3.html+css
